# GitHub Description

## Short Repo Description
Local-first chargeback evidence desk for small merchants: triage disputes, find evidence gaps, and draft response packets.

## Longer Description
Chargeback Evidence Desk turns Stripe, Shopify, PayPal, Square, or generic dispute exports into a local evidence queue. It maps messy CSV columns, parses pasted dispute text, flags missing evidence, ranks cases by deadline and amount at risk, and exports Markdown/JSON packet outlines.

## Alternative Descriptions
1. Local browser app for prioritizing chargeback disputes and assembling evidence packet outlines.
2. Small-business dispute triage desk with flexible CSV mapping, deadline scoring, and evidence gap detection.
3. Turn messy processor dispute exports into a local chargeback action queue and Markdown evidence report.
